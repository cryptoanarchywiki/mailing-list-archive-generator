SecureDrive works under Windows 95 if you disable all 32-bit disk drivers. 

Control Panel
System
Performance
File System
Troubleshooting
Disable all 32 bit protected mode disk drivers

You have to run SECTSR before starting Win95.  A pain but it does work.
