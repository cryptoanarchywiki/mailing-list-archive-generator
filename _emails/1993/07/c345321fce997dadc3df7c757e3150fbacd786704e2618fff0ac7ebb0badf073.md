>I just fetched macpgp 2.3 from soda, and wound up with a file ending in
>the suffix "gz". This is a new one on me, and I don't know how to decode

has been made, but the source code is available. However, I have the prog
available here at aql.gatech.edu /pub/cypherpunks/pgp/binaries in cpt.hqx
format. The source code for gzip is also available in /pub.
