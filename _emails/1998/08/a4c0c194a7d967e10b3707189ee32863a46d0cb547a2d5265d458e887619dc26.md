E-mail Security Alert
=====================

Recently Microsoft was notified by AUSCERT (http://www.auscert.org.au), OUSPG (http://www.oulu.fi) and NTBugtraq (http://ntbugtraq.ntadvice.com) of a security issue affecting the way Microsoft e-mail clients handle file attachments with extremely long file names.

On July 27th, Microsoft published a patch for Outlook 98 and Outlook Express 4.x that fixes the problems as reported. For more information about this problem and how to fix it, link to the following Microsoft security bulletin:

   http://www.microsoft.com/security/bulletins/ms98-008.htm

Microsoft has published the following two Knowledge Base (KB) articles on this issue:

   - Microsoft Knowledge Base (KB) article Q168019, Update Available For Outlook Express Security Issue,
http://support.microsoft.com/support/kb/articles/q168/0/19.asp

   - Microsoft Knowledge Base (KB) article Q175807, OL98:Update Available For Outlook 98 Security Issue,
http://support.microsoft.com/support/kb/articles/q175/8/07.asp
